# RigForce

Webapp React + Three.js per autorig di modelli `.glb/.gltf` direttamente nel browser. Permette di generare scheletri per umanoidi, quadrupedi, uccelli e robot usando bounding box e di esportare un file JSON con le posizioni delle ossa.

## Requisiti
- Node 18+

## Avvio
```bash
npm install
npm run dev
```
